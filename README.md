# Java Fundamentals Projects

This repository contains a collection of small Java projects created to practice and strengthen core Java programming skills.

The goal of these projects is to build a solid foundation in **Java programming and Object-Oriented Programming (OOP)** through hands-on implementation.

---

## Topics Covered

The projects in this repository focus on practicing important Java concepts such as:

- Classes and Objects
- Constructors
- Encapsulation
- Arrays
- Arrays of Objects
- Loops and Iteration
- Conditional Statements
- Method Design
- Basic Data Structures
- Program Organization

---

## Projects

This repository will continue to grow as more projects are added. Examples include:

- Coffee Inventory System
- Student Management System
- Library Management System
- Maze Solver
- Array Practice Programs

Each project is placed in its own folder and contains the Java classes needed to run the program.

---

## Purpose of This Repository

The purpose of this repository is to:

- Practice Java fundamentals through small projects
- Develop problem-solving skills using Java
- Build a portfolio of Java programming work
- Prepare for more advanced topics such as cloud development and large-scale systems
